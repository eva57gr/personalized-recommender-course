# hands-on-recommender-system

# Dependencies

- [Python v3.11](https://www.python.org/downloads/)
- [uv v0.4.30](https://github.com/astral-sh/uv)

# Install

```bash
make install
```

# Usage

### Run Streamlit app

```bash
make run-app
```

### Clean Hopsworks resources

```bash
make clean-hopsworks-resources
```
